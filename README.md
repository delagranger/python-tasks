#Python Tasks
Этот репозиторий создан для отслеживания моего прогресса в освоении языка Python.
Освоение происходит посредством решения задач на различные темы, начиная с основ языка.

В репозиторий выкладываются тех. задание и реализация. Обе конструкции находятся по соседству, в отдельной функции.
Под каждую задачу создается собственная функция.

Задачи сгруппированы по файлам, каждый из которых соответствует определенной теме.

Основной источник задач на данный момент - курсы на Stepik.
1. ["Поколение Python": курс для начинающих](https://stepik.org/course/58852/syllabus)
2. ["Поколение Python": курс для продвинутых](https://stepik.org/course/68343/syllabus)
